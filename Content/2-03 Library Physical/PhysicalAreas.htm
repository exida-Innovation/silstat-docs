Areas

The Areas library allows for creation and configuration of Areas including Process Areas, Units, Equipment, and more. These Areas can be used to create your Plant Hierarchy.

Select the Areas library to view all Areas in the database. A grid shows the following properties for each.

  • Type
  • Name
  • Status
  • Parent Name
  • Plant Type
  • Process Type
  • Labels
  • Action

 

 

To create a new Area, select the New button on the top right-hand corner of the library view. This will open the Area view. To View or Edit an Area, select the Edit button in the Action column. You can input the following properties for the Area.

 

Property

Description

Area Types

Types include Business, Country, Region, Site, Facility, Plant, Storage, Unit, Process Area, Equipment. Some selections will add properties to your Area. Units will allow you to enter Plant Types and Process Types. Equipment Areas will allow you to enter Node Types. These types and subtypes can be defined by the user in the SILstat Settings.

Name

This can be entered as text.

Description

This can be entered as text.

Comments

This can be entered as text.

Location

Locations can be added to the Location library and associated with an Area.

Assigned Procedures

Procedures can be added to the Procedures Library and associated with an Area.

 

The Lifecycle Section shows all events associated with the Area.

The Hierarchy section shows all Parents and Children associated with the Area. These could be other Areas, Hazards or Tags.

The Common Properties section shows all Resources, References and Labels associated with the item.

To create a new item, select the Create button in the top right-hand corner. Items that are not created will be saved as a draft.

The Insights on the Area view show SIF Safeguards, Tags, Procedures, Operating Hours, Demands, and Failures.

To delete the item, select the Delete button in the top right-hand corner. After deleting the item, it becomes a soft delete. A soft delete means the item can be recovered with the help of exida for a period of 180 days. After that period is over, the item will become permanently deleted.

Unit

Property

Description
Type Automatically selected as Unit.
Name This can be entered as text.
Description This can be entered as text.
Comments This can be entered as text.
Location Locations can be added to the Location library and associated with an Area.

The Operations section shows Plant Type, Process Type, Process Mode, and Process Description.

The Lifecycle section shows all events associated with the Area. To view more information on what each state means, select the Legend button.

The Hierarchy section shows all the Parents and Children associated with the Unit. Children can be Areas and Tags.

The Common Properties section shows all Resources, References and Labels associated with the item.

 

Equipment

Property

Description
Type Automatically selected as Unit.
Name This can be entered as text.
Description This can be entered as text.
Comments This can be entered as text.
Location Locations can be added to the Location library and associated with an Area.

The Assigned Procedures section shows the Procedures that have been directly assigned to this node. To add a new Procedure, select the New button. For more information on how to create a Procedure, see section X.X. To associate with an existing Procedure, select the Associate button.

The Lifecycle section shows all events associated with the Area. To view more information on what each state means, select the Legend button.

The Hierarchy section shows all the Parents and Children associated with the Equipment. Parents include Units, and Children include Hazards and Tags.

The Common Properties section shows all the Resources, References, and Labels associated with the item.

To delete the item, select the Delete button in the top right-hand corner. An item will not be able to be deleted if it is in service.

After deleting the item, it becomes a soft delete. A soft delete means the item can be recovered with the help of exida for a period of 180 days. After that period is over, the item will become permanently deleted.

The Insights on the Equipment view show Devices, Hazards, Failures, and Operating Hours.

To record an event against the Equipment node, select the Record button to record an Install, Decommission, or Bypass event.
