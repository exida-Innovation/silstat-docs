Admin Center

The Admin Center allows a user with an administrator role to define users and user roles for the SILstat environment.

To navigate to the Admin Center, select the SILstat on the top bar of the Home Page. This will open a drop-down menu that allows you to switch from SILstat to Dashboard, Licensing, Admin Center or Help and Support. Select ‘Admin Center’ to proceed.

The Admin Center Homepage shows your recent activity within the Admin Center. You can select an item from the ‘Recent’ list to return to that item’s view. You will also see information on your Organization, exida Training and Help, and exida News.
