Notifications

You will be able to receive in-app notifications, including:

  • Welcome Messages
  • Announcements of New Releases and Help documentation
  • Approval Workflow
    • Approval Required
    • Approval Accepted/Rejected
  • Notifications about Imports

You will also receive notifications via email. These include:

  • Welcome Message
  • Approval Workflow
    • Approval Required
    • Approval Accepted/Rejected

A Notification icon is located at the top of the SILstat Home Page, showing a red dot when a Notification is available. When you hover over the icon, the number of Notifications is displayed. Select the Notification icon to view your notifications. Select ‘View’ to see more details. Select the Delete icon to dismiss the Notification.
