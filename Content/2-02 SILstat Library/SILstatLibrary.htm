SILstat Library

The SILstat Library provides a centralized location for the data in your SILstat database. A library item is a unique entity that can be referenced/linked in multiple locations. Items that have been added to the plant hierarchy can be found here, as well as items that are in draft. A change to the library item will automatically be applied to all locations the library item is referenced/linked. Using libraries will dramatically increase the efficiency and consistency of the various work activities to be performed. To navigate to the Library, select the Library button on the left side panel. At the top you will see your most frequently accessed libraries listed. Under your Frequent libraries, you will see all available libraries listed.
